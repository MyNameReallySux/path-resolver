    PathResolver

    Class that creates a suite of methods useful for resolving paths within a project. Includes a directory resolver, which creates relative path resolvers for each directory in a given directory map. Support for resolver renaming and alias generation, which formats the resolver name to reflect the alias as the root rather than the full path.
